# Developers-Lunchbox
Developers-Lunchbox
Overview The Developer Lunchbox is a playful showcase project that bundles together three mini‑tools inside a single “lunchbox” layout:

🎲 Dice Roller — roll any number of dice and see results with images.

🔑 Password Generator — create secure random passwords with customizable character sets.

💵 Currency Converter — convert ZAR into GBP, USD, EUR, or JPY using fixed exchange rates.

📝 Task Manager - can manage tasks with the following functions, add, filter, delete, and mark as completed.

The idea is to present these utilities like compartments in a lunchbox — each tool tucked neatly into its own section with a zig‑zag layout.

Features Responsive design with custom CSS and Google Fonts.

Smooth reveal animations when clicking View.

Interactive JavaScript logic for dice, password generation, currency conversion and a task manager.

Mobile‑friendly layout with @media queries for iPhone 12 Pro and similar devices.

Installation & Usage Clone the repo:

bash git clone https://github.com/Nathidev123/developer-lunchbox.git Open index.html in your browser.

Explore the lunchbox compartments:

Roll dice 🎲

Generate passwords 🔑

Convert currencies 💵

Task Manager📝

Authenticity Note: This project started as a mix of tutorial code‑alongs and my own experimentation. I used tutorials to learn the baseline logic (like dice rolling and password generation), but then customized and expanded it into my own creative concept — the “Developer Lunchbox.”

I also collaborated with AI tools for troubleshooting, layout refinement, and presentation polish. The core coding, creative vision, and persistence are mine; AI was used as a sparring partner, not a crutch. I believe in being transparent about my process, because learning is about remixing, experimenting, and making ideas your own.

Future Improvements Fetch live currency exchange rates via an API.
